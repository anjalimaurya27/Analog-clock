# 🕒 Analog Clock

A simple **Analog Clock** built using **HTML, CSS, and JavaScript**.  
This project displays the current time with moving **hour, minute, and second hands**.

The clock updates every second using JavaScript's **Date object**.


## 🚀 Features
- Real-time analog clock
- Smooth movement of clock hands
- Simple and clean UI
- Beginner-friendly project


## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
  

## ▶️ How to Run

1. Clone or download the repository
2. Open **index.html** in your browser
3. The analog clock will start automatically
